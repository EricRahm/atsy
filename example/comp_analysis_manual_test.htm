MemBench

Note: This is based a copy of Gregor Wagners membench test

This benchmark automatically opens 30 sites with a 10 second pause after each new site. A textbox appears 60 seconds after the last page is loaded indicating that the benchmark has finished.
You can see detailed information about your memory behavior if you type about:memory in your Firefox URL bar.

You will most likely have to disable the popup blocker for this site.
